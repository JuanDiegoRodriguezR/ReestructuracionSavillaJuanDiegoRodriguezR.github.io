    PROCESO DE RECUPERACION DE MATERIALES

    Recoleccion

    Es una de las partes mas interesantes y curiosas del proceso porque no sabemos que nos vamos a encontrar

    Transporte

    Dichos materiales encontrados los llevamos a una de nuestras sedes en la cual poder iniciar el proceso de recuperacion

    Recuperacion y Reincoporacion

    Porceso el cual se hace limpieza del material y eliminacion de partes dañadas

    Aprovechamiento

    Parte donde se mira el que hacer con dichos materiales, de como darles una nueva vida y se util para las personas

    Montaje

    Proceso donde ya teniendo una idea clara se lleva acabo, se arma, pinta, se deja reluciente para la venta

    Venta

    Es nuestro proceso final para nosotros, pero el inicio de una utilidad para nuestros productos
